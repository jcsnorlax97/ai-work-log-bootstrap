# AI Work Logs

This is a private, Obsidian-compatible work-log vault for AI coding-assistant session captures and generated daily summaries.

Do not publish this vault to a public repo.

## Folder Layout

```text
inbox/YYYY/MM/DD/conversations/
inbox/YYYY/MM/DD/notes/
generated/YYYY/MM/YYYY-MM-DD.md
templates/
hooks/
```

## Daily Flow

1. Save redacted session captures into `inbox/YYYY/MM/DD/conversations/`.
2. Add manual notes into `inbox/YYYY/MM/DD/notes/`.
3. Generate daily summaries into `generated/YYYY/MM/YYYY-MM-DD.md`.
4. Review generated notes before copying or linking them into another vault.

## Security

- Do not store secrets or raw private data here.
- Keep raw transcript capture disabled unless explicitly needed for low-risk sessions.
- Treat generated tags as candidates until reviewed.
